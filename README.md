# Courier Management System

## Description

A Database Management System (DBMS) project developed to manage courier services, shipments, customers, and deliveries efficiently.

## Features

* Customer Registration
* Courier Booking
* Shipment Tracking
* Delivery Management
* Employee Management
* Payment Records

## Technologies Used

* MySQL
* PHP
* HTML
* CSS
* JavaScript
* XAMPP

## Database Tables

* Customers
* Couriers
* Shipments
* Employees
* Payments

## Installation

1. Install XAMPP.
2. Import the SQL file into MySQL.
3. Place the project folder in `htdocs`.
4. Start Apache and MySQL.
5. Open the project in your browser.
